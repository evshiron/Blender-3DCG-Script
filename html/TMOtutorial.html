TMOファイルを出力する時、「3dcg_tmo_export.py」を使うのですが、 ボーンのHeadの位置を動かさないで下さい。
親を変えたり、ボーンのTailの位置を変えたり、コンストレインツを使うことはできます。
よって、リギングである程度ポーズを付けやすくする事ができます。

3dcg_tmo_export.py 自体の使い方は簡単ですが、モーションを付けれるようになるまでに手間が掛かります。
・アーマチュアをTSOファイルから取り込む。
・ボーンだけでは表情を付けにくいのでボディを取り込んだり、スカートや髪も必要になるかもしれません。
・仕様により、Armatureを二つ用意する必要があります。
・IKを使うための設定をする。
等…


まず標準ボディTSOファイル(N001BODY_A00.tso 等)から、アーマチュアとメッシュを取り込みます。
File→ Import→ 3DCG tso file (.tso)...
MeshとArmatureにチェックを入れる。
ファイルを選択する。
Importを押せば、現在のレイヤーに追加されます。


扱いやすくする為に、メッシュオブジェクトを[Ctrl+J]で統合します。


TSOファイルのノードには初期値が設定されており、3dcg_tso_import.pyで取り込んだアーマチュアにもRotが適用されています。
このアーマチュアにアーマチュアモデファイアを設定すると、実際とは違う形に変形されます。
しかし、出力する時には初期値が必要になるので、もう一つ同じアーマチュアを用意して、 元のアーマチュアの動きを完全にコピーさせる事になります。

まず[Shift+D]でコピーします。
コピーをポーズモードにして、[Apply Pose as Restpose]を押します。
おまけのスクリプト、「ArmatureMimic」を使ってコピーを元のアーマチュアの動きを真似させるようにします。


メッシュオブジェクトにアーマチュアモデファイアを追加します。
コピーのアーマチュアのオブジェクト名を入力します。


適当にポーズを付けます。


開始フレームと終了フレームの設定をします。 どちらも同じ値の場合は、ただのポーズになります 3dcg_tmo_export.pyを起動します。
File→ Export→ 3DCG tmo file (.tmo)...


設定し、元のアーマチュアを選択し[Export]ボタンを押します。


TAHにして3DCGに持ってきました。 手を上げただけのポーズですが、適用されている事がわかります。
